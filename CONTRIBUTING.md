# Contributing

Quelora's contributing guide is centralized for the whole project — one place,
kept current, covering every package.

👉 **https://github.com/Quelora/quelora/blob/main/CONTRIBUTING.md**

It explains how to spin up a local environment (`quelora-dx-env`), what kinds
of contributions are welcome, and how to propose changes. Bug reports and pull
requests specific to **this** package are very welcome right here.
